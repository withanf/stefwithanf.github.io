Background

A designer, a cheapskate, and a fast learner walk into a bar.

She orders a stout.

I needed a portfolio website. Website builders like Squarespace and Wix are too limiting. Framer and Webflow are handy, but not necessarily useful outside of this context. On top of that, these options were expensive because of features I didn't need (e.g., CMS). I had a microsite that was slow-loading with bad UX. So how might I launch my portfolio in a way that's fast, effective, and cheap?

My success criteria:

  • Fully populated with content
  • Responsive
  • Adheres to WCAG 2.0 AA standards
  • Improved performance and SEO

Site development

I have HTML and CSS experience to begin with thanks to college, MySpace, and Neopets, but it was surface-level. I found several great resources, including the tutorials on Interneting is Hard to learn beyond the basics. Once I felt I had a solid foundation, I decided on Bootstrap to accelerate development. As for hosting, I already had a Github account thanks to the game jams I participate in, and Github Pages are free with SSL built in.
